DEMO VIDEO INSTRUCTIONS
========================

This file is a placeholder for the demo-video.mp4 file.

To create the actual demo-video.mp4 file:

1. RECORDING OPTIONS:
   - Use screen recording software (OBS Studio, Camtasia, QuickTime, etc.)
   - Record your screen showing the portfolio website
   - Include voice narration explaining features

2. VIDEO CONTENT SHOULD INCLUDE:
   - Introduction (10-15 seconds)
   - Navigation demonstration
   - Responsive design showcase (desktop, tablet, mobile views)
   - Each section walkthrough:
     * Hero section
     * About section
     * Skills section
     * Projects section (hover effects, links)
     * Contact form demonstration
   - Mobile menu functionality
   - Scroll animations
   - Conclusion (5-10 seconds)

3. VIDEO SPECIFICATIONS:
   - Duration: 3-5 minutes recommended
   - Resolution: 1080p (1920x1080) or 720p (1280x720)
   - Format: MP4 (H.264 codec)
   - Audio: Clear narration with background music (optional)

4. TOOLS FOR CREATION:
   - OBS Studio (Free, open-source)
   - Camtasia (Paid)
   - QuickTime (Mac)
   - Windows Game Bar (Windows 10/11)
   - Loom (Online, free tier available)
   - Screencast-O-Matic (Free/Paid)

5. EDITING (Optional):
   - Add text overlays
   - Include transitions
   - Add background music
   - Trim unnecessary parts

6. SAVE THE FILE:
   - Export as MP4
   - Name it: demo-video.mp4
   - Place it in the presentation/ folder

NOTE: For submission purposes, you can also create a text file named 
demo-video.mp4 with a link to a YouTube/Vimeo video if file size is an issue.
